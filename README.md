# RailSense: Enhanced Train Tracker

RailSense is a modern, high-performance train tracking and analytics dashboard. It provides real-time insights, station status, and journey planning with a premium user interface.

## How to Run Locally (Easiest Method)

Follow these simple steps to get RailSense running on your computer:

### 1. Get the Files
- **Download the Project**: Click the green **"Code"** button on this GitHub page and select **"Download ZIP"**.
- **Extract the ZIP**: Once downloaded, find the file (usually in your Downloads folder), right-click it, and select **"Extract All..."**. Choose a folder where you want to keep the project.

### 2. Run the App
- **Open the Folder**: Open the extracted folder.
- **Launch index.html**: Find the file named `index.html`. 
- **Right-click `index.html`** and select **"Open with"**, then choose your favorite web browser (like **Google Chrome**, **Microsoft Edge**, or **Firefox**).

That's it! The app should now be running in your browser.

## Technical Features
- **Real-time Search**: Quickly find trains and stations.
- **Interactive Maps**: View train routes and live positions.
- **PNR Tracking**: Check your ticket status easily.
- **Responsive Design**: Works beautifully on desktops and tablets.
